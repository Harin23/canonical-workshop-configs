# Canonical Workshop

This repository houses the declarative configuration files for managing a sandboxed, hardware-accelerated development workspace using [Canonical Workshop](https://canonical.com/blog/introducing-workshop-sandboxed-development-environments).

The primary focus of this setup is running a local LLM server inside an isolated system container, while seamlessly exposing its API endpoints back to the host machine for consumption by modern editors like **Zed**.
